# 🛕 TTD Website – Tirumala Tirupati Devasthanam

A responsive and interactive frontend website project based on the famous Tirumala Tirupati Devasthanam (TTD) temple. This project provides information about temple history, services, darshan timings, image gallery, and an online ticket booking interface.

## 📌 Project Overview

This website is designed using **HTML, CSS, and JavaScript** with a modern UI and responsive layout. It includes multiple sections such as:

- Home Page
- Temple History
- TTD Information
- Temple Timings
- Services Section
- Tirumala Image Gallery
- Online Darshan Ticket Booking Form

The website also includes animations, background video support, responsive design, and form validation.
